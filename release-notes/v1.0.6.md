## Deutsch

- **Sprachen:** Menü **Einstellungen → Sprachen** — UI auf **Englisch**, **Deutsch** oder **Portugiesisch (Portugal)**; Speicherung in **`visustruct.properties`** (`uilanguage`: `en`, `de`, `pt_PT`).
- **I18n:** Menüleiste und ausgewählte Dialoge über **ResourceBundles**; **Korrektur:** gewähltes **Englisch** wird nicht mehr durch die **System-Locale** (z. B. Deutsch) überschrieben (`ResourceBundle.Control` ohne Fallback auf `Locale.getDefault()`, **`Messages_en.properties`**).
- **Build:** Maven **`de.visustruct:visustruct:1.0.6`**, Fat-JAR **`visustruct-1.0.6.jar`**, Release-Anhang zusätzlich **`visustruct.jar`**.

## English

- **Languages:** **Settings → Languages** — UI in **English**, **German**, or **European Portuguese (Portugal)**; stored in **`visustruct.properties`** (`uilanguage`: `en`, `de`, `pt_PT`).
- **I18n:** Menu bar and selected dialogs use **resource bundles**; **fix:** **English** UI is no longer overridden by the OS **default locale** (dedicated control + **`Messages_en.properties`**).
- **Build:** Maven **`de.visustruct:visustruct:1.0.6`**, artifact **`visustruct-1.0.6.jar`**, release also ships **`visustruct.jar`**.
