## Deutsch

- **Zeichenfläche:** Farben über **`CanvasStyle`** an die helle VisuStruct-Palette (Hintergrund, Kanten, Markierung, Drag/Vorschau).
- **Speichern:** Reiter-Titel und „bearbeitet“-Markierung (`*`) beziehen sich immer auf das **richtige** Diagramm (nicht nur auf den aktiven Tab); leere Reitertitel werden abgefangen (kein Absturz mehr beim Speichern). XML-Schreiben in **UTF-8**, bei Fehlern erscheint eine **Meldung** (statt stiller Konsolenausgabe).
- **macOS:** Menüleiste **im Fenster** (`apple.laf.useScreenMenuBar=false`) — zuverlässiges **File → Save** mit **FlatLaf**; Verhalten wie unter **Windows** und **Linux**. Nach **Tab umbenennen** wird der Speicherdialog erst im **nächsten UI-Takt** geöffnet (Dialog bleibt nicht „hängen“). **FlatLaf** auf **3.7.1** aktualisiert.
- **UI (hell):** Explizite Farben für **MenuItem** / **Checkbox-** und **Radio-Menüeinträge** in **`VisuStructTheme`** (lesbar statt „alles grau“). Beim Überschreiben einer existierenden Datei ist das Hauptfenster **Parent** des Bestätigungsdialogs.
- **Build:** Maven **`de.visustruct:visustruct:1.0.5`**, Fat-JAR **`visustruct-1.0.5.jar`**, Release-Anhang zusätzlich **`visustruct.jar`**.

## English

- **Canvas:** Colors unified via **`CanvasStyle`** (background, borders, selection, drag/preview).
- **Saving:** Tab title and dirty `*` are updated for the **correct** diagram (not only the selected tab); empty tab titles are handled safely. XML output uses **UTF-8**; failures show an **error dialog**. 
- **macOS:** Menu bar **inside the window** for reliable **Save** with **FlatLaf** (same pattern as **Windows** / **Linux**). **Rename tab** defers the save dialog with **`invokeLater`**. **FlatLaf** bumped to **3.7.1**.
- **UI (light theme):** **`VisuStructTheme`** sets **MenuItem** / checkbox / radio menu item colors. **Overwrite** confirmation uses the main frame as **parent**.
- **Build:** Maven **`de.visustruct:visustruct:1.0.5`**, artifact **`visustruct-1.0.5.jar`**, release also ships **`visustruct.jar`**.
