# VisuStruct 1.0.5 — release notes

- **Canvas:** Colors unified via **`CanvasStyle`** (background, borders, selection, drag/preview).
- **Saving:** Tab title and dirty `*` are updated for the **correct** diagram (not only the selected tab); empty tab titles are handled safely. XML output uses **UTF-8**; failures show an **error dialog**.
- **macOS:** Menu bar **inside the window** for reliable **Save** with **FlatLaf** (same pattern as **Windows** / **Linux**). **Rename tab** defers the save dialog with **`invokeLater`**. **FlatLaf** bumped to **3.7.1**.
- **UI (light theme):** **`VisuStructTheme`** sets **MenuItem** / checkbox / radio menu item colors. **Overwrite** confirmation uses the main frame as **parent**.
- **Build:** Maven **`de.visustruct:visustruct:1.0.5`**, artifact **`visustruct-1.0.5.jar`**, release also ships **`visustruct.jar`**.
