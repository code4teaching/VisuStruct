# VisuStruct 1.0.7 — release notes

### Source code generator

- **Delphi/Pascal** removed; export is **Java** or **Python** only.
- **Python (3.10+):** multi-way branch as **`match` / `case` / `case _:`**; other constructs use colon + indentation (`if`/`else`, `for`, `while`, `while True`, do-while style via `while True` + `if not (…):` + `break`).
- **Generate code** dialog: target language and options are **localized**; choice is stored in **`visustruct.properties`** as **`celanguage`** (`0` = Java, `1` = Python).
- **Comments in generated code:** Java `/* … */`, Python `""" … """`.

### Structure chart / API

- **Multi-way branch:** default column label is **`default`** (Java keyword), not “Else”; string from **`structure.multiway.defaultCaseLabel`** (i18n).
- **`XMLLeser`:** public load methods are named **`ladeXML`** (fixes the old typo `ladeXLM`).

### Cleanup

- Removed unused **selection state** and **size-box** leftovers (TODO/upstream) from **`Struktogramm`** / **`StruktogrammElement`**.

### Build

- **Maven:** **`de.visustruct:visustruct:1.0.7`**, artifact **`visustruct-1.0.7.jar`**, release also ships **`visustruct.jar`**.
